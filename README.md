# PThreads PS3

[POSIX](https://pthreads-emb.sourceforge.net) for embedded systems (PTE) ported to PS3

### End user
```shell
make docker-build
```
Then get the static library at build-cmake/libpthread.a

### For developers
Build
```shell
make cmake-build
```

Install
```shell
make cmake-install
```
Uninstall
```shell
make cmake-uninstall
```
